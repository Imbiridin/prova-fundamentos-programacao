Descrição: Sistema simple para registro de vendas de uma loja.

Como executar:
python main.py

Etapa 1: 
Iniciar o repositório com -git init-
Selecionar os arquivos do repositório com -git add .-

Etapa 2:
iniciar o sistema de vendas no main.py
criei as variáveis vazias para armazenar os valores a medida que o usuário for inserindo os dados
em seguida  criei um loop com o while. A medida que o usuário for inserindo os dados o programa vai armazenando
com o If coloquei as condições para as opções do sistema, o desconto, dependendo do valor da venda.

Etapa 3:
Finalizei o programa.
